# Solarized python matplotlib colormap
## What is it?
Very simple matplorlib colormaps based on the solarized color palette by [Ethan Schoonover](http://ethanschoonover.com/solarized).

## Requirements
 - matplotlib
 - numpy

## Usage
```python
import solarized
Sol = solarized.Solarized()
cmap = Sol.cmap()
```
